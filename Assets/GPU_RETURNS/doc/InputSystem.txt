アクションマップを利用したInputSystemの利用方法

１．コードの先頭に以下の命令を追加して、InputSystemを利用可能にする
　　using UnityEngine.InputSystem;

２．アクションを保存するメンバ変数を用意する
    InputAction stickL;                 // アクションマップ【F310_LStick】を取得
	InputAction buttonA;				// アクションマップ【F310_A】を取得

３．アクションマップファイルから（InputSystem_Actions）アクションを取得して変数に保存
	void Start()
    {
        // アクションマップを取得
        stickL = InputSystem.actions.FindAction("F310_LStick");
        buttonA = InputSystem.actions.FindAction("F310_A");	
	}
	
４．アクションを利用して入力制御を行う
	void Update()
    {
		// Stickの入力情報
		if (stickL.ReadValue<Vector2>().magnitude != 0)
		{
			Debug.Log("左Stick = " + stickL.ReadValue<Vector2>().ToString());
		}
        if (buttonA.IsPressed())
        {
            Debug.Log("Aボタン(Zキー)が押されている");
        }
		
	}

※ GPU_RETURNSシステムでは、ロジクールのゲームパッドF310とキーボードが連動するアクションが用意されている。
